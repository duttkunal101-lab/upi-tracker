# Card artwork drop-in folder

Put a card image here and it **automatically** replaces the rendered card on the
site — no code change needed. Use these exact file names (PNG, ideally a
landscape card image ~1000×630, transparent or solid background both fine):

| File name to upload | Card |
|---|---|
| `ace.png`        | Axis Bank ACE Credit Card |
| `flipkart.png`   | Flipkart Axis Bank Credit Card |
| `atlas.png`      | Axis Bank Atlas Credit Card |
| `airtel.png`     | Airtel Axis Bank Credit Card |
| `myzone.png`     | Axis Bank MY Zone Credit Card |
| `neo.png`        | Axis Bank NEO Credit Card |
| `insta-easy.png` | Axis Bank Insta Easy (against FD) |

A custom **bank logo** can go one level up as `axis/assets/axis-logo.png`.

If a file is missing, the site falls back to its built-in card design — so you
can add them one at a time. JPG/WebP also work if you tell us the exact name.
